================================================================================
CONCEPTOS CLAVE
================================================================================

QUE ES BETA?

  Beta mide SENSIBILIDAD de un activo al mercado
  
  Formula: R_activo = Alfa + Beta Ã— R_mercado + Error
  
INTERPRETACION:

  Beta > 1   = Mas volatil que el mercado (Agresivo)
  Beta = 1   = Igual que el mercado (Neutral)
  Beta < 1   = Menos volatil (Defensivo)
  Beta < 0   = Se mueve opuesto (Hedging)

USOS PRACTICOS:

  1. Control de riesgo: Seleccionar activos con beta deseada
  2. Diversificacion: Combinar activos con diferentes betas
  3. Cobertura: Usar betas negativas para hedging
  4. CAPM: Estimar retornos esperados
  5. Performance: Comparar vs benchmark
